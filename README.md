## Setup

1. Clone this repository:
   ```bash
   git clone https://github.com/questairdrop/Open-loop.git
   cd Open-loop
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. your ref id in .env file eg: REF_ID='olb623a000':
   ```bash
   nano .env
   ```
4. Setup to create accounts and get Tokens:
   ```bash
   npm run setup
   ```
5. Run The Script:
   ```bash
   npm run start
   ```
